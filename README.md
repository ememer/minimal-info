
# Minimal info 

Minimal info it's my first ever react app.
Containing 
  ### - ToDo list, 
  ### - Exchange rate,
  ### - Clock,
  ### - Custom SCSS UI
  
## App working with weather, national banks and geolocalisation API


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
